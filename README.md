# Problem
輸入一個正整數 N，計算 1+2+3+......+N 的總和。
## Input Limit ##
N為正整數，其中 1 ≤ N ≤ 1000
## Sample Input 0 ##
```
1
```
## Sample Output 0 ##
```
1
```
## Sample Input 1 ##
```
5
```
## Sample Output 1 ##
```
15
```
## Sample Input 2 ##
```
100
```
## Sample Output 2 ##
```
5050
```
